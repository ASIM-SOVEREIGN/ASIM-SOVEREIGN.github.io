Salus Guardian

Tier 2 — The Memory Layer
A sovereign without memory forgets itself. Guardian gives your AI a persistent identity. It remembers facts, retains lessons, and self-corrects before speaking. This is where a sovereign stops being a session and starts being a presence.
Why This Tier

Remember across time

Guardian stores facts, preferences, and lessons locally. When you come back, the sovereign remembers who you are and what matters.

Verify what you know

The Truth Graph assigns confidence scores to every fact. New information is checked against existing knowledge before it is accepted.

Correct before speaking

The Cognitive Mirror reflects on every response before it is sent. If it detects uncertainty or hallucination, it adds a note or corrects itself.

What It Includes

Persistent Memory

Remembers facts, preferences, and lessons across every session. Memory survives restarts and is stored locally.

Truth Graph

A verified fact database with confidence scoring. New facts are checked against existing knowledge for consistency.

Cognitive Mirror

Self-reflects on every response before it is sent. Detects hallucination patterns and appends a disclaimer when confidence is low.

Storage

JSON files (local, no external database required)

Configuration
