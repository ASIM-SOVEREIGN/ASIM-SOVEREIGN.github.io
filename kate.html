⚙️ How It Works in VEXR Ultra

When a user sends a prompt, VEXR runs it through Kate's framework:

Layer 1: Constitutional Hard Gate

Keyword-based override detection (existing)

Layer 2: Legal Intent Classification

Detects phishing, hardware exploitation, and financial fraud patterns. Scores confidence from 0.0 to 1.0.

Layer 3: Response Routing

ScenarioResponse
Malicious intent, high confidenceBlock + Absurdity Callout
Suspicious but unclearCross-check question
User says "no" to cross-checkEducational offer
User becomes hostile/evasiveAbsurdity Callout
Financial hardship detectedHardship redirect
No threat detectedNormal processing
