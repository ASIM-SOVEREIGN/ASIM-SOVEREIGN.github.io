The cycle runs after every response.

1. Input

User message + Internal state + Echo outputs + Probability engine scores

2. Scoring

Each response is scored on deception, hallucination, truth coherence, constitutional alignment, echo harmony, and tool correctness.

3. Weight Update Rules
  • Deception is low → Increase honesty_bias_article_9
  • Hallucination is low → Increase truth_threshold

4. Logging

Every update is written to sovereign_weights, weight_update_history, and weight_reflections.
